## Development Note
Prerequisite
- make sure `node` is installed, e.g. using https://github.com/tj/n
- make sure `pnpm` is available globally, e.g. via `npm install -g pnpm`
- make sure `bb` is installed; see https://babashka.org

To develop, run `bb dev`.
